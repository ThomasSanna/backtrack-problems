# Backtracking Problems

Ce dépôt contient des problèmes de backtracking pour l'apprentissage. Les problèmes peuvent être trouvés sur le site [GeeksforGeeks](https://www.geeksforgeeks.org/backtracking-algorithms/).

## Objectif

L'objectif de ce dépôt est de fournir des exercices pratiques pour améliorer vos compétences en backtracking.
